Course Syllabus

This is the syllabus for our Intro to Web Development course.

Course Description

In this course, you will learn the basics of web development, including HTML, CSS, and JavaScript. We will cover how to create and style web pages, as well as how to add interactivity using JavaScript.

Course Schedule

  • Week 1: Introduction to HTML
  • Week 2: Introduction to CSS
  • Week 3: Introduction to JavaScript
  • Week 4: Building a Simple Website
  • Week 5: Final Project
