# Cyb 100 - Intro to Web Development Course Website

A comprehensive course website for the **Intro to Web Development** course, featuring course materials, assignments, syllabus, and resources for students learning HTML, CSS, and JavaScript.

## 📋 Course Overview

**Cyb 100** is a foundational web development course designed to introduce students to the fundamentals of building websites. The course covers the essentials of front-end web development and provides a structured curriculum with hands-on assignments.

**Instructor:** Abdulrahman Amusat

### Course Topics
- Introduction to HTML - Structure and semantic markup
- Introduction to CSS - Styling and layout
- Introduction to JavaScript - Interactivity and dynamic behavior
- Building websites - Combining all three technologies
- Final project - Creating a complete, functional website

### Course Schedule
- **Week 1:** Introduction to HTML
- **Week 2:** Introduction to CSS
- **Week 3:** Introduction to JavaScript
- **Week 4:** Building a Simple Website
- **Week 5:** Final Project

## 🎨 Design & Styling

The website uses a clean, modern design with the following features:

- **Font:** Raleway (imported from Google Fonts)
- **Color Scheme:**
  - Dark green containers with azure text
  - Light grey background
  - Accent colors (red, orange, green)
- **Layout:** Centered, responsive flexbox design
- **CSS Classes:**
  - `.app` - Main app container with flexbox centering
  - `.container` - Content wrapper with padding and shadow
  - `.bg-dark-green` - Dark green background
  - `.bg-grey` - Light grey background
  - `.text-red`, `.text-green`, `.text-orange` - Text color utilities
  - `.text-italics` - Italic text styling

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser to view the home page
3. Navigate through the different sections using the provided links
4. Explore the course materials and assignments

- Any modern web browser (Chrome, Firefox, Safari, Edge)

- Pure HTML, CSS, and JavaScript
